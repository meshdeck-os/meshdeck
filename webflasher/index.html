How to flash

  1. Plug the T-Deck into this computer with a USB-C data cable and switch it on.
  2. Click Flash MeshDeck above, pick the serial port, choose Install.
  3. Wait ~1 minute. The device reboots into MeshDeck when done.
Device not showing up? Put it in download mode: switch it off → hold the trackball button in → switch on while holding → release, then click Flash again. Power-cycle once after flashing.

After flashing

  • Set your name: Settings → Node name, then send an advert.
  • On first boot: enter a node name, then pick a radio preset (default EU/UK Narrow 869.618 / BW 62.5 / SF 8 / CR 8).
  • Optional: put uk.mdm in /meshdeck-maps/ on an SD card for the high-detail map.
  • Phone app still works — pair over Bluetooth, PIN 123456.

Update later from an SD card (no computer)

  1. Download the latest firmware (your browser saves it as firmware.bin).
  2. Copy it to the root of a FAT32 SD card and insert the card.
  3. On the device: Settings → Update firmware from SD. It flashes and reboots.
First install must use the flasher above. SD updates only work once MeshDeck is already running — a bare device has nothing on it yet to read the card.
